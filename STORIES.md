# User stories

Userstory 1 - Bok katalog
Som användare vill jag kunna se en lista med böcker så att jag kan välja vilka böcker jag är intresserad av. 

Userstory 2 - Favorit böcker
Som användare vill jag kunna favoritmarkera böcker så jag kan spara de i mina böcker.

Userstory 3 - Ta bort favoritmarkering
Som användare vill jag kunna ta bort en favoritmarkering så jag ta bort det ifrån mina böcker när jag läst klart.

Userstory 4 - Lägga till böcker
Som användare vill jag kunna lägga till en ny bok ifall den inte finns med på läslistan och önskar läsa den.

Userstory 5 - Statistik sidan
Som användare vill jag kunna öppna statistik sidan så jag kan se information om böckerna.
